# mstrie

## Description
The implementation of the `mstrie` program that is used for the Multiset-trie data structure management.

---

## Installation
To install `mstrie` use the following commands:
```bash
./configure
make
make install
make clean
```

---

## Uninstallation
To uninstall `mstrie` use the following commands:
```bash
make uninstall
```
